# 🚀 React Chapter 1 – Fundamental Concepts

## 📖 Giới thiệu
Đây là project học tập được xây dựng bằng **ReactJS**, tổng hợp các kiến thức nền tảng quan trọng như:
- **Interpolation**
- **Bind Attribute**
- **Method**
- **Raw HTML**
- **Event Handling**
- **Two-way Binding**
- **useMemo (Computed)**
- **Conditional Rendering**
- **List Rendering**

Project được xây dựng nhằm mục đích **thực hành và củng cố kiến thức React cơ bản**.
---
## 🛠 Công nghệ sử dụng

- **React** (Functional Component)
- **React Hooks** (`useState`, `useMemo`)
- **Bootstrap 5**
- **CSS**
---
